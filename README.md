# Construction DPR Dashboard

Construction DPR Dashboard with PDF parsing, labour analytics, and Excel export.

## Features

* Upload Daily Progress Report PDFs
* Automatic data extraction
* Contractor-wise labour tracking
* Skilled & Unskilled labour analysis
* Activity and quantity monitoring
* Remarks extraction
* Search and filtering
* Excel export

## Tech Stack

### Frontend

* React
* Vite
* CSS

### Backend

* FastAPI
* Python

### Libraries

* pdfplumber
* pandas
* openpyxl

## Project Overview

This application automates the extraction of Daily Progress Report (DPR) data from PDF files and presents it through an interactive dashboard for construction project monitoring.
